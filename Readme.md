#NetWorm

Данная программа моделирует распространение эпидемии в неоднородных сетях.
Под эпидемией может пониматься распростраение сетевых вирусов в сети интернет.

##Инструкция:

1. Загружаем сеть (в виде матрицы степени вершин графа - матрица связности графа)
2. Выбираем модель распространение эпидемии (SI, SIER и т.д.)
3. Заполняем вероятности переходов состояний узлов (матрица переходов) 
4. Запускаем эпидемию. 

Написано на С++/Qt4.8,Qwt.
